== Description ==

Share Your Posts on India's Own Social Media "Wrauter" 

Service Link(Wrauter): https://wrauter.com
Service Terms of Use : https://wrauter.com//terms/terms

Features:

** This is a very simple plugin that share your wordpress posts to Wrauter 
** Wordpress blog authors can place these shortcodes anywhere to share his content to wrauter.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin in WordPress > Plugins > wrauterwp Social Login > Activate

And You are done everything
